# Fuzzy-System-Prediksi-Pemberian-KIP
KIP adalah salah satu program pemerintah untuk membantu pemerataan pendidikan di Indonesia. Penerapan Fuzzy System dalam Prediksi Pemberian KIP bagi Mahasiswa ini merupakan salah satu bentuk implementasi AI untuk menentukan mahasiswa-mahasiswa yang berhak mendapatkan KIP.
